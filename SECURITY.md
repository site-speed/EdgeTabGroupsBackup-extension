# Security Policy — EdgeTabGroupsBackup

## Reporting a vulnerability

Please report security issues privately via the public repository’s security policy.

## Scope

- The extension operates locally and uses standard Edge/Chromium extension APIs.
- Backups are stored in your bookmarks/favourites.
- Folder cleanup is user-confirmed and only attempts to delete a selected bookmarks folder after exact-name confirmation.
- Edge may block deletion of Workspace-managed folders.

## Supported versions

Only the latest published version is supported.
