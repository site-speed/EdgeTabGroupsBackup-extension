# Privacy Policy — EdgeTabGroupsBackup

_Last updated: 2026-05-14_

## Summary

EdgeTabGroupsBackup backs up your current Edge window’s tabs and tab groups into your favourites (bookmarks) as folders and links.

## Data handling

- **No network / backend**: The extension does not send your data to any server.
- **Local-only**: All processing happens inside your browser.
- **What it accesses**:
  - Tab URLs and titles (to save as bookmarks).
  - Tab group titles (to name folders).
  - Bookmarks/favourites (to create backup folders and bookmark entries, and to attempt deletion of selected folders when you explicitly use the cleanup feature).

## Storage

- The extension stores only minimal configuration in extension storage (e.g. maximum backups to keep).

## Contact

If you discover a security or privacy issue, please report it via the security policy in the public repository.
