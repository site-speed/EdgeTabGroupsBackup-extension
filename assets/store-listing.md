# Microsoft Edge Add-ons Store Listing — EdgeTabGroupsBackup

## Product name

EdgeTabGroupsBackup

## Short description

Back up the current Edge window’s tabs and tab groups into favourites/bookmarks.

## Store description

EdgeTabGroupsBackup helps you save a bookmark-based backup of the tabs and tab groups in the current Microsoft Edge window.

Use it to create timestamped backup folders in your favourites/bookmarks, either under an Edge Workspace favourites folder or under the extension’s configurable default destination in Other favourites / Other bookmarks.

Key features:

- Back up the current window’s normal web tabs into favourites/bookmarks.
- Preserve tab group structure as bookmark subfolders.
- Choose a Workspace folder or the configurable default backup destination.
- Preview total, ungrouped, grouped, and unsaved-tab counts before backup.
- Keep only the newest N backups per destination with configurable retention.
- Optionally pin a marker tab to add a predictable label to backup folder names.
- Handle InPrivate windows with a fixed In-Private label and disabled marker controls.
- Clean up backup folders from Options where Edge allows deletion.

The extension runs locally in the browser. It does not send tab or bookmark data to a backend service.
