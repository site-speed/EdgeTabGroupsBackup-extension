EdgeTabGroupsBackup — Options

Retention and maintenance settings for backups.

Backup settings

The default folder is always listed in the popup, but it is only created the first time you use it as the backup destination. If you rename it here, an existing folder with the old name is renamed where possible.

Workspace folder cleanup

Best-effort cleanup for workspace folders and the default backup folder under Other favourites. Workspace cleanup removes matching backup subfolders first; Edge may still block deletion of Workspace-managed empty folders.
